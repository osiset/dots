# Dot files

## Setup

+ DE: XFCE ~Plasma~
+ WM: i3
+ Bar: Polybar
+ Notification Manager: Dunst
+ Network Manager: NetworkManager (nmtui)
+ Runner: Rofi (drun, window)
+ Bluetooth: Blueberry
+ Audio: PulseAudio
+ Greeter: LightDM
+ Clipboard: Clipit
+ Shell: ZSH (Oh-My-ZSH)
+ Terminal: URXVT (font: Hack)
+ Colors: Scorched & Crackpot (from Rainglow.io)
